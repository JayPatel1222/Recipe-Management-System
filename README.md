# Recipe Management System

## Overview
The Recipe Management System is a full-stack web application built with ASP.NET MVC, C#, Entity Framework Core, and SQL Server.  
This project was developed as a **team project** during my Software Development diploma at New Brunswick Community College.  
It allows users to create, manage, and browse recipes along with their ingredients and categories. The project demonstrates full-stack development skills, database design, and responsive UI development.

## Features
- **User Dashboards:** Manage personal recipes in a structured way.  
- **CRUD Operations:** Create, Read, Update, Delete recipes, categories, and ingredients.  
- **Responsive Design:** Built with Bootstrap to ensure mobile-friendly layouts.  
- **Search & Filter:** Quickly find recipes by name or category.  
- **Image Uploads:** Secure server-side handling of recipe images.  
- **Validation:** Server-side input validation to prevent invalid or malicious data.  
- **MVC Architecture:** Clear separation between frontend, backend, and database layers.

---

## Technologies Used
- **Frontend:** HTML, CSS, Bootstrap, Razor Pages  
- **Backend:** C#, ASP.NET MVC, Entity Framework Core  
- **Database:** SQL Server  
- **Tools & Practices:** Git, Visual Studio, Agile workflow, MVC architecture

---

## Architecture
- **Database:** Relational schema for Recipes, Categories, and Ingredients.  
- **Backend:** Controllers handle HTTP requests, implement business logic, and interact with the database through Entity Framework Core.  
- **Frontend:** Razor Pages render dynamic content and manage user interactions; Bootstrap ensures responsiveness.  

---

## Installation & Setup
1. Open the project in Visual Studio.  
2. Restore NuGet packages.  
3. Update `appsettings.json` with your SQL Server connection string.  
4. Run database migrations to create the schema.  
5. Press F5 or click **Run** to start the application locally.
